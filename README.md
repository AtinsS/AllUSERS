# Поиск пользователей

Проект по поиску и отображению данных пользователей, созданный в рамках тестирования функционала классического React.

## 🔧 Функционал

- **Поиск пользователей** по следующим критериям:
  - Имя
  - Email
  - Специальность
- **Отображение данных** пользователей в удобном формате
- **Фильтрация** результатов поиска в реальном времени

## 🛠 Технологии

- **React** - JavaScript библиотека для построения пользовательских интерфейсов
- **CSS** - Стилизация компонентов
- **TypeScript** - Основной язык разработки

## 🚀 Запуск

```
git clone
cd ALLUSERS
npm run dev
```
